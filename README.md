##  📌 Study Rule
- 매일 한 문제 필수
  - 필수 문제 1, 선택 문제 1
- 월수금 9-11시 온라인
  - 월 : 평가 코드 공유 및 리뷰
  - 수 : 화수 코드 리뷰
  - 금 : 목금 코드 리뷰
  - 주말 : 유동적으로 문제 제공 및 풀이

## Git Commit Message
|*Type*|*Subject*|
|:---|:---|
|**[PLATFORM]**|문제 해결|
|**[FIX]**|수정|
|**[DOCS]**|문서 (문서 추가, 수정, 삭제)|
|**[CHORE]**|빌드, 설정 파일|

**ex) [PLATFORM] 문제번호. 문제이름 `git commit -m '[BOJ] 1001. A+B'`**
